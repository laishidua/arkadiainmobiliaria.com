== Installation ==

1. Upload the entire `coming-soon-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to coming soon wp menu to customize this plugin.
4. Finally Enjoy it.
